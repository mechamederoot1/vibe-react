📱 Como Visualizar o App:

🚀 Opção 1: Metro Bundler (Atual)

O Metro bundler está rodando na porta 8081.

npm start

Use emulador ou dispositivo físico conectado.

📱 Emulador Android: npm run android
🍎 Emulador iOS: npm run ios
📲 Expo (QR Code): npm run expo
🌐 Versão Web: npm run web

✅ Funcionalidades Implementadas:

🔐 Sistema de Login/Cadastro
📝 Criação de Posts e Depoimentos
📖 Stories com Editor Avançado
👤 Perfil Completo do Usuário
🎨 Design Nativo Responsivo

💡 Para melhor experiência, use um emulador ou dispositivo físico
